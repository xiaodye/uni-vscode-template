## **简述一下**这个模板给 VSCode 开发` uni-app`带来的体验

- 增强`pages.json`和`manifest.json`开发体验（语法提示、颜色块、写注释）
- 一键创建页面、组件、分包
- 完善的`API`，组件，uni.scss 语法提示
- 条件编译注释高亮
